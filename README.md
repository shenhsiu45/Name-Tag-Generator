# Name Tag Generator

這個專案涉及在React.js中建立一個互動式的名牌生成器，允許創建和刪除一組人的批量名牌。

## Objective
* 建立一個數據模型。
* 將功能組件轉換為有狀態的組件。
* 通過建立文本輸入使其具有互動性。
* 處理用戶輸入，以便它可以創建新的名牌。
